# VigilGuard Documentation

Welcome to the official documentation site for VigilGuard.

## About VigilGuard

VigilGuard helps organizations understand and control their external exposure.
The platform brings together attack-surface discovery, continuous monitoring, risk insights, and vendor security tracking.
